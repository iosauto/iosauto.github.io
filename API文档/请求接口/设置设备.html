设置设备

每一个设备通过airplay连接到投屏服务后都需要进行usb鼠键设备设置和设备参数设置才能正常使用
usb鼠键设备的pid和vid可通过获取已连接USB设备列表里得到
设备参数可以通过获取支持设备类型列表里得到

请求参数​

{
  "fun": "set_dev",
  "msgid": 0,
  "data": {
    "mac": "{{mac}}",
    "vid": "520A",
    "pid": "000A",
    "width": "414",
    "height": "896",
    "device_name": "iPhone XS Max",
    "version": "14.0.1",
    "location": "33|179|750|23|86|0|17|48|687|13|29|888|7|9|199|5|5|179|1|1|600",
    "gid": "123456",
    "name": "1111"
  }
}
参数名示例值参数类型是否必填参数描述
funset_devString是功能名
msgid0Number是消息id,服务端接收后会原样返回，websocket连接可用用来异步转同步用。http模式直接填0
data-Object是参数内容
data.mac00:00:00:00:00String是设备mac地址
data.vid520AString否usb设备vid
data.pid000AString否usb设备pid
data.width414String否手机屏幕真实宽
data.height896String否手机屏幕真实高
data.device_nameiPhone XS MaxString否手机型号
data.version14.0.1String否手机系统版本
data.location-String否usb设备鼠标参数
data.gid123456String否分组id
data.name1111String否设备名(用户自定义名称)

成功响应示例​

{
  "fun": "set_dev",
  "msgid": 0,
  "data": {
    "state": 0,
    "ip": "192.168.31.239",
    "mac": "76:90:DE:1F:12:2A",
    "vid": "520A",
    "pid": "000A",
    "width": "414",
    "height": "896",
    "device_name": "iPhone XS Max",
    "version": "14.0.1",
    "location": "33|179|750|23|86|0|17|48|687|13|29|888|7|9|199|5|5|179|1|1|600",
    "gid": "123456",
    "name": "1111"
  },
  "status": 0,
  "message": "成功"
}
参数名示例值参数类型参数描述
funset_devString功能号
msgid0Number消息id,服务端接收后会原样返回，websocket连接可用用来异步转同步用。http模式直接填0
data-Object返回数据
data.state-Number0不在线，非0在线
data.ip192.168.31.239String设备ip地址
data.mac76:90:DE:1F:12:2AString设备mac地址
data.vid520AStringusb设备vid
data.pid000AStringusb设备pid
data.width414String手机屏幕真实宽
data.height896String手机屏幕真实宽
data.device_nameiPhone XS MaxString手机型号
data.version14.0.1String手机系统版本
data.location-Stringusb设备鼠标参数
data.gid123456String分组id,如果为
data.name1111String设备名(用户自定义名称)
status0Number0成功,非0失败
message成功String调用返回消息

data.mac76:90:DE:1F:12:2AString设备mac地址
status1Number0成功,非0失败
